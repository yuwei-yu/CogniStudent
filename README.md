# CogniStudent

辅导员辨识学生比赛/训练桌面程序，使用 PySide6 + qt-material 构建 Windows GUI，可通过 GitHub Actions 打包为单文件 exe。

## 环境配置

- Python 3.9+
- Windows 打包建议使用 Python 3.11
- 安装依赖：

```bash
pip install -r requirements.txt
```

本地运行：

```bash
python src/main.py
```

## 数据目录

程序运行目录下固定使用 `resources/` 作为运行时数据根目录。首次启动会自动创建：

- `resources/admin_config.json`
- `resources/contest_settings.json`

默认管理员账号为 `admin`，密码为 `admin123`。如需修改，直接编辑 `resources/admin_config.json`。

活动目录必须位于 `resources/` 下，例如：

```text
resources/
└── 测试辅导员大赛/
    ├── 张三-105018.xls
    ├── 张三-105018/
    │   ├── 3191621001.jpg
    │   └── 3191621002.png
    ├── 李四-205052.xls
    └── 李四-205052/
```

每名辅导员由一个同名 Excel 文件和一个同名照片文件夹组成，命名格式为 `姓名-工号`。

## Excel 字段

程序会校验并读取以下中文字段：

`学号`、`姓名`、`专业`、`政治面貌`、`担任职务`、`家庭住址`、`宿舍`、`家庭经济状况`、`心理健康状况`、`英语四六级`、`不及格科目`、`奖惩情况`

照片文件名应为 `学号.jpg`、`学号.jpeg` 或 `学号.png`。照片缺失会警告，但不阻止导入。

## 使用流程

当前版本按单机双屏比赛流程运行，不再提供辅导员登录和评委登录。管理员登录后完成活动、名单和比赛设置，然后启动比赛控制台。

- 创建、删除活动
- 上传辅导员和学生资料 zip
- 下载上传模板
- 勾选并添加本次比赛参赛辅导员到 `contest_config.json`
- 设置所有比赛统一使用的计时、抽题数量、答题信息字段
- 启动比赛控制台，按参赛名单顺序逐位辅导员答题
- 一键导出全部历史数据 zip，便于备份或恢复完整 `resources` 数据
- 删除辅导员 Excel 和照片文件夹

比赛时建议连接两个屏幕：

- 答题屏：给辅导员看题目、照片或线索
- 评委屏：给评委看标准信息和答案
- 控制台可选择每个窗口所在屏幕，也可选择该屏幕显示“题目”或“标准信息”
- 控制台支持缩放、开始/暂停/重置计时、上一位/下一位参赛辅导员

## 比赛设置

管理员页面提供默认比赛设置，设置对所有活动统一生效：

- 大海捞针计时：默认 180 秒
- 鱼目混珠计时：默认 120 秒
- 描述定位计时：默认 120 秒
- 大海捞针抽取人数：默认 3 人
- 鱼目混珠本人学生数：默认 2 人
- 鱼目混珠干扰照片数：默认 8 张
- 描述定位题数：默认 2 题
- 答题字段：默认包含姓名、专业、政治面貌、担任职务、家庭住址、宿舍、家庭经济状况、心理健康状况、英语四六级、不及格科目、奖惩情况

设置保存在 `resources/contest_settings.json`。

## 数据备份与恢复

本程序不依赖数据库，运行数据都在 `resources/` 目录内。

常规比赛：

1. 创建活动并上传辅导员资料。
2. 勾选参赛辅导员，点击“添加参赛辅导员”。
3. 设置比赛计时和答题字段，点击“保存比赛设置”。
4. 进入活动详情，点击“启动比赛”。
5. 在比赛控制台为答题屏和评委屏选择显示器、显示内容和缩放比例。

历史数据备份：

1. 管理员首页点击“一键导出全部数据”，生成完整历史数据 zip。
2. 下次启动或换电脑时，管理员登录后可选择导入历史数据 zip。
3. 导入后，本机 `resources` 会恢复对应活动、配置等文件。

导出的 zip 文件使用 UTF-8 文件名，跨 Windows/macOS 解压时中文路径应保持正常。

## 上传模板

根目录的 `template.zip` 是用户可见模板。管理员界面点击“下载上传模板”会优先复制根目录模板；如果根目录模板不存在，会使用打包进 exe 的 `src/resources/template.zip` 后备模板。

上传资料 zip 可包含一个或多个辅导员，每个辅导员必须包含：

- `姓名-工号.xls` 或 `姓名-工号.xlsx`
- `姓名-工号/` 照片文件夹

程序会防止 zip 路径遍历，并在同名辅导员存在时询问覆盖或跳过。

## Windows 打包

本项目已配置 `.github/workflows/build.yml`，使用 `windows-latest` 自动打包：

```powershell
pyinstaller --onefile --windowed --name "CogniStudent" `
  --paths src `
  --add-data "src/resources/template.zip;resources/template.zip" `
  --collect-all qt_material `
  src/main.py
```

打包产物会上传为 `CogniStudent-Windows` artifact。

## 常见问题

- Excel 列名错误：请按模板字段命名，或使用程序已兼容的常见别名，例如“学生姓名”“宿舍号”等。
- `.xls` 读取失败：确认安装了 `xlrd>=2.0`。
- 照片不显示：确认照片位于同名辅导员文件夹内，文件名与学号完全一致。
- 没有活动可选：使用管理员登录并创建活动，或手动在 `resources/` 下放置活动文件夹。
- 第二块屏幕不显示：先在系统显示设置中确认已连接扩展屏，再在比赛控制台选择对应屏幕并点击“显示双屏”。
